Colophon
========

This site is built on plain old semantic `HTML5`_,
powered by the `rstblog`_ static site generator for `Python`_,
and styled using `CSS3`_,
with help from `jQuery`_,
`Email Defuscator`_,
`IcoMoon`_,
`Breakpoint`_,
`Modular Scale`_,
`Susy`_,
`Compass`_,
`Sass`_,
and several `GreenMud`_ fonts.
